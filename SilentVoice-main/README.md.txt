# Silent Voice

## Requirements
- Python 3.x

## Setup
1. Create virtualenv:
   - `python -m venv silentvoice`
2. Activate it:
   - Windows: `.\silentvoice\Scripts\activate`
   - macOS/Linux: `source silentvoice/bin/activate`
3. Install deps:
   - `pip install -r requirements.txt`
4. Run:
   - set FLASK_APP=main
   - set FLASK_DEBUG=1
   - flask run
